Ai Enrollement Manager project task 2 based on Iron lady Company

The Iron Lady Enrollment Manager is a high-fidelity administrative command center designed for managing elite female leadership programs. It moves beyond simple data entry by combining robust CRUD (Create, Read, Update, Delete) functionality with Google Gemini AI to provide strategic foresight. The application features a premium, brand-aligned aesthetic using a Deep Maroon and Cream palette, tailored for executive-level operations.
